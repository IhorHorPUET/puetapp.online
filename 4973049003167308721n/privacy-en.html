Use of Google Calendar API

The "Розклад занять в ПУЕТ" application integrates with Google Calendar using the Google Calendar API to allow users to add and remove information about their class schedules directly to their personal Google Calendar.

Limited Scope of Access:

The Application only requests permission to add and remove events specifically related to the user's class schedule in their Google Calendar.

The Application does not have the ability to view, read, or modify any other events or personal information stored in the user's Google Calendar.

No Access to Personal Data:

The Application does not access, collect, or store any personal information from the user's Google Calendar other than the specific schedule events that the user has chosen to add or remove through the Application.

The Service Provider does not have access to any personal events or data stored in the user's Google Calendar that are unrelated to the class schedule created through the Application.

User Control:

Users have full control over the integration with Google Calendar. They can choose to grant or revoke the Application's access to their Google Calendar at any time.

The Application only interacts with the specific events that the user has created through the Application, ensuring that the user's other calendar data remains private and secure.
